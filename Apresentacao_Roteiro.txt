=== SLIDE 1: Visão Geral do Projeto ===
- Título: GolPro - Sistema de Gerenciamento Esportivo
- O que é: Um sistema C# via Console para gerenciar campeonatos de futebol.
- Entidades Principais: 
  - Time (Código, Nome, Cidade, Estatísticas)
  - Jogador (Matrícula, Nome, Time, Gols)
  - Partida (Mandante, Visitante, Data, Placar)
- Objetivo: Prover CRUD das entidades e calcular automaticamente a classificação geral e a artilharia do campeonato.

=== SLIDE 3: Conceito de POO 1 - Encapsulamento ===
- Título: Protegendo nossos Dados (Encapsulamento)
- Explicação: No GolPro, os dados das entidades não podem ser alterados de qualquer jeito. Todos os atributos são `private` e só podem ser lidos/alterados de forma segura pelas propriedades `public`.
- Trecho de Código (Cole no slide):
```csharp
private int _golsMarcados;

public int GolsMarcados
{
    get { return _golsMarcados; }
    set { _golsMarcados = value; } // Opcional: adicionar validações aqui no futuro
}
```

=== SLIDE 4: Conceito de POO 2 - Construtor Parametrizado e Sobrecarga ===
- Título: Inicialização de Objetos
- Explicação: Utilizamos dois tipos de construtores. O padrão (vazio) para criar instâncias "zeradas" e o parametrizado para instanciar as Models já com dados vindos dos nossos arquivos .txt.
- Trecho de Código (Cole no slide):
```csharp
// Construtor Parametrizado
public TimeModel(string codigo, string nome, string cidade)
{
    _codigo = codigo;
    _nome = nome;
    _cidade = cidade;
    _pontos = 0; // Inicializa zerado
}
```

=== SLIDE 5: Conceito de POO 3 - Composição / Injeção de Dependência ===
- Título: Controllers Trabalhando Juntos
- Explicação: A classe `PartidaController` não tenta fazer tudo sozinha. Ela recebe (`composição / agregação`) instâncias prontas de `TimeController` e `JogadorController` para conseguir consultar times e atribuir gols.
- Trecho de Código (Cole no slide):
```csharp
private TimeController _timeController;
private JogadorController _jogadorController;

// Recebe as dependências no construtor
public PartidaController(int col, int row, int width, int height, Tela tela, 
                         TimeController timeCtrl, JogadorController jogCtrl)
{
    _timeController = timeCtrl;
    _jogadorController = jogCtrl;
}
```
